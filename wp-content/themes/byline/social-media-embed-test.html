Features

  • Automatic Detection: Recognizes social media links when pasted
  • Instant Embedding: Converts links to embedded widgets automatically
  • Visual Feedback: Shows notification when embedding occurs
  • Multiple Platforms: Supports Facebook, Twitter/X, YouTube, TikTok, Instagram, LinkedIn, and Reddit
  • Responsive Design: Embeds adapt to different screen sizes
  • Works Everywhere: Functions in both regular posts and LiveBlog content blocks

Technical Details

  • File: wp-content/themes/byline/assets/js/social-media-embedder.js
  • Integration: Automatically loaded on post and LiveBlog edit pages
  • Dependencies: jQuery, TinyMCE (for rich text editing)
  • Browser Support: Modern browsers with clipboard API support
  • Performance: Lightweight with minimal impact on page load
